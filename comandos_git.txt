git init . -> cria um repo git na pasta atual
git status > mostra o status do trabalho
git add > adiciona arquivo para ser comitado
git commit -m "msg" > faz commit do que estava para ser commitado
git log > mostra todos os históricos de commit


##FLUXO DE TRABALHO
1. criar o diretório (pasta) do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome do arquivo>
6. git status
7. git commit -m "msg"
8. git status
9. bolta para o passo 3.
